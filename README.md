# 📊 Clutch

Профессиональный бэкенд-сервис для создания динамических реляционных таблиц, вдохновленный платформой Airtable. Проект позволяет пользователям на лету конструировать базы данных, управлять кастомными типами полей (включая сложные структуры данных через JSONB) и гибко настраивать отображение данных.

---

## ✨ Ключевые возможности

*   **Динамическая схема данных:** Создание таблиц, строк и колонок через API без необходимости пересборки приложения.
*   **Поддержка JSONB (PostgreSQL):** Хранение гибких метаданных и конфигураций ячеек (например, в таблице `clutch`).
*   **Управление миграциями:** Строгий контроль схемы данных с помощью Liquibase.
*   **Расширяемые типы полей:** Поддержка строк, чисел, дат и кастомных JSON-объектов.

---

## 🛠️ Стек технологий

*   **Язык и фреймворк:** Java 21 / Spring Boot
*   **База данных:** PostgreSQL 15+ (активное использование типа `jsonb`)
*   **Миграции:** Liquibase (XML-чейнджсеты)
*   **Контейнеризация:** Docker / Docker Compose

---

## 🚀 Быстрый старт

### 1. Предварительные требования
Убедитесь, что у вас установлены:
*   Docker & Docker Compose
*   JDK 21 или выше
*   Ваш любимый сборщик (Maven / Gradle)

### 2. Запуск базы данных
Поднимите локальный контейнер PostgreSQL:
```bash
docker-compose up -d
```

### 3. Запуск приложения
```bash
# Для Maven:
./mvnw spring-boot:run

# Для Gradle:
./gradlew bootRun
```
Приложение запустится на порту `8080`.

---

## 🗄️ Особенности архитектуры базы данных

Проект использует гибридный подход: классические реляционные таблицы совмещены с неструктурированными JSONB-полями для обеспечения «бессхемной» гибкости.
